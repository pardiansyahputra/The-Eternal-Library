# Berikut tata cara setup Macro & VBA dari Developer mode

###### Materi: penjelasan umum terkait lab reverse shell

###### Media: docx/docm + cloudflareTunneling

---

**penjelasan**
hal ini penting dilakukan sebab di word awal kita tidak akan menjumpai tab Developer, nah berikut beberap langkah langkah singkat yang bisa saya paparkan kepada anda:

**cara menampilkan Tab Developer**

- buka aplikasi `ms-word`
- masuk ke Tab file dan pilih `setting`
- dipanel kiri masuk ke bagian `customize ribbon`
- dibagian `Main Tabs` pilih dan centang `Developer`
- jangan lupa tekan oke/save

**cara untuk nulis macro di MS-Word**

- masuk ke tab `developer`
- masuk ke `visual basic`
- pilih tab insert `didalam visual basic nya`
- pilih `module` dan masukkan kodenya
